왜 주식 ETF를 별도 분석하나요?
ETF는 주식에 속한 자산만 추종합니다. 주식 ETF와 채권 ETF를 같이 분석하면 기초자산의 성격 차이(위험·수익 특성)가 결과를 왜곡합니다. 단거리 선수와 마라토너의 기록을 섞어 평균 내면 의미가 없듯이, 분류체계 내에서 비교해야 의미 있는 통계가 나옵니다.

왜 혼합자산 ETF를 별도 분석하나요?
ETF는 혼합자산에 속한 자산만 추종합니다. 주식 ETF와 채권 ETF를 같이 분석하면 기초자산의 성격 차이(위험·수익 특성)가 결과를 왜곡합니다. 단거리 선수와 마라토너의 기록을 섞어 평균 내면 의미가 없듯이, 분류체계 내에서 비교해야 의미 있는 통계가 나옵니다.

왜 기타 ETF를 별도 분석하나요?
ETF는 기타에 속한 자산만 추종합니다. 주식 ETF와 채권 ETF를 같이 분석하면 기초자산의 성격 차이(위험·수익 특성)가 결과를 왜곡합니다. 단거리 선수와 마라토너의 기록을 섞어 평균 내면 의미가 없듯이, 분류체계 내에서 비교해야 의미 있는 통계가 나옵니다.

기술통계 (Descriptive Statistics)

☕ 커피숍 비유: 하루 매출을 매일 기록했을 때 — 평균은 '보통 얼마 벌었나', 표준편차는 '들쭉날쭉한 정도'입니다.
  • 평균(Mean): 모든 값의 합 ÷ 개수. 극단값에 민감
  • 중앙값(Median): 순서상 가운데. 극단값 영향 없음
  • 표준편차(Std): 평균에서 얼마나 퍼져있나. 클수록 변동성 큼
  • 왜도(Skewness): 분포의 비대칭성. 양수=오른쪽 꼬리 길다
  • 첨도(Kurtosis): 분포의 뾰족함. 양수=극단값 많음
평균: x̄ = Σxᵢ/n
표준편차: s = √[Σ(xᵢ-x̄)²/(n-1)]
🔗

상관분석 (Pearson Correlation)

🌡️ 온도와 아이스크림 판매량: 온도가 오를수록 판매량도 오르면 r ≈ +1. 관계의 강도와 방향을 측정합니다.
  • r = +1: 완벽한 양의 선형관계
  • r = 0: 선형관계 없음
  • r = -1: 완벽한 음의 선형관계
  • |r| > 0.7: 강한 상관, 0.3~0.7: 중간, <0.3: 약한 상관
  • p < 0.05: 우연이 아닌 통계적으로 유의한 상관관계
r = Σ[(xᵢ-x̄)(yᵢ-ȳ)] / √[Σ(xᵢ-x̄)² × Σ(yᵢ-ȳ)²]
t = r√(n-2)/√(1-r²), df = n-2
📈

단순회귀 (Simple Regression)

🎯 키와 몸무게: 키가 1cm 클수록 몸무게가 평균 몇 kg 더 나가는지 직선으로 표현합니다.
  • Ŷ = b₀ + b₁X: b₁=기울기, b₀=절편
  • R²: X가 Y 변동의 몇 %를 설명하나 (0~1)
  • R²=0.1: X 혼자 Y의 10% 설명. ETF에서는 충분히 의미있음
  • t검정: 기울기 b₁이 0이 아닌지 (= X가 Y에 영향 있는지)
b₁ = Σ[(xᵢ-x̄)(yᵢ-ȳ)] / Σ(xᵢ-x̄)²
b₀ = ȳ - b₁x̄
R² = 1 - SS잔차/SS합계
🔢

다중회귀 (Multiple Regression)

🏠 집값 예측: 면적·층수·지역·방수를 동시에 고려해서 집값을 예측하는 것처럼, 여러 독립변수를 동시에 분석합니다.
  • Ŷ = β₀ + β₁X₁ + β₂X₂ + ... + βₖXₖ
  • 수정R²(Adjusted R²): 변수 수를 보정한 설명력
  • F검정: 모형 전체의 유의성 (모든 β=0이 아닌지)
  • 각 β의 t검정: 다른 변수를 통제했을 때 개별 변수의 효과
  • 다중공선성: 독립변수들이 서로 높은 상관이면 계수 불안정
β = (XᵀX)⁻¹Xᵀy
F = (SS회귀/k) / (SS잔차/(n-k-1))
수정R² = 1-(1-R²)(n-1)/(n-k-1)
📏

t-검정 (Independent t-test)

⚖️ 두 학교 시험 성적 비교: A학교 vs B학교의 평균 성적 차이가 우연인지 진짜 차이인지 판단합니다.
  • 귀무가설 H₀: 두 집단 평균이 같다 (μ₁ = μ₂)
  • Levene 검정: 등분산 여부 먼저 확인 (p>0.05 → 등분산)
  • 등분산 t: t = (x̄₁-x̄₂)/(Sp·√(1/n₁+1/n₂))
  • 이분산(Welch) t: 자유도 보정 후 t검정
  • p < 0.05: 두 집단 간 유의한 차이 있음
Sp² = [(n₁-1)s₁² + (n₂-1)s₂²] / (n₁+n₂-2)
t = (x̄₁-x̄₂) / (Sp√(1/n₁+1/n₂))
📐

ANOVA (일원분산분석)

🏫 세 학교 이상 성적 비교: t검정은 2개 집단만 비교하지만, ANOVA는 3개 이상을 동시에 비교합니다. 사후검정으로 어느 쌍이 다른지 확인합니다.
  • SS분해: SS합계 = SS처리(집단간) + SS오차(집단내)
  • F비: MS처리/MS오차. 집단간 변동이 집단내 변동보다 크면 F>1
  • η²(에타제곱): 집단 구분이 전체 분산의 몇 %를 설명하나
  • p < 0.05: 적어도 한 집단 평균이 다름
SS처리 = Σnⱼ(ȳⱼ-ȳ)², SS오차 = ΣΣ(yᵢⱼ-ȳⱼ)²
F = (SS처리/df처리) / (SS오차/df오차)
η² = SS처리/SS합계
🔢

풀이 공식 & 엑셀 활용법

각 분석 방법의 수학 공식과 Excel 데이터 분석 도구 사용법

📊 기술통계 — 공식 & 엑셀
n : 표본 크기
평균: x̄ = (1/n)Σᵢxᵢ
분산: s² = Σ(xᵢ-x̄)²/(n-1)
표준편차: s = √s²
중앙값: 정렬 후 가운데 값
사분위수 범위(IQR) = Q3 - Q1
왜도: γ₁ = (n/((n-1)(n-2))) Σ((xᵢ-x̄)/s)³
Excel 방법:
① 데이터 탭 → 데이터 분석 → 기술 통계법
② 입력 범위: Y열 선택, 첫 행 레이블 체크
③ 요약 통계량 체크 → 출력 범위 지정
🔗 피어슨 상관분석 — 공식 & 엑셀
r = Σ[(xᵢ-x̄)(yᵢ-ȳ)] / √[Σ(xᵢ-x̄)² · Σ(yᵢ-ȳ)²] = Cov(X,Y) / (Sₓ · Sᵧ)

검정통계량: t = r·√(n-2) / √(1-r²)
자유도: df = n-2
H₀: ρ = 0 (모상관계수 = 0)
|r|의 해석: 0.1~0.3 약, 0.3~0.7 중, 0.7+ 강
Excel 방법:
① 데이터 분석 → 상관 분석
② 입력 범위: Y와 X 전체 선택
③ 상관행렬 출력 → 각 셀의 값이 r
④ p값 계산: =T.DIST.2T(ABS(r*SQRT(n-2)/SQRT(1-r^2)), n-2)
📈 단순선형회귀 — 공식 & 엑셀
모형: Ŷ = b₀ + b₁X

최소제곱 추정:
b₁ = Σ(xᵢ-x̄)(yᵢ-ȳ) / Σ(xᵢ-x̄)²
b₀ = ȳ - b₁·x̄

결정계수: R² = SS회귀/SS합계 = 1 - SS잔차/SS합계
SS합계 = Σ(yᵢ-ȳ)²
SS잔차 = Σ(yᵢ-Ŷᵢ)²
SS회귀 = SS합계 - SS잔차

b₁의 표준오차: Sᵦ₁ = √(MSE/Σ(xᵢ-x̄)²)
t = b₁/Sᵦ₁, df = n-2
Excel 방법:
① 데이터 분석 → 회귀 분석
② Y입력 범위: 수익률 열
③ X입력 범위: 분석할 독립변수 열
④ 잔차 체크 → 출력 확인 (회귀통계, 분산분석표, 계수표)
🔢 다중선형회귀 — 공식 & 엑셀
모형: Ŷ = β₀ + β₁X₁ + β₂X₂ + ... + β₇X₇
변수: X₁추적오차, X₂=ln(순자산), X₃괴리율, X₄총보수,
X₅외국인비중, X₆=ln(거래대금), X₇존속기간

행렬 표현: β̂ = (XᵀX)⁻¹Xᵀy

수정R² = 1 - (1-R²)(n-1)/(n-k-1)
F = (R²/k) / ((1-R²)/(n-k-1))

각 계수의 t검정:
SE(β̂ⱼ) = √(MSE·[(XᵀX)⁻¹]ⱼⱼ)
t = β̂ⱼ/SE(β̂ⱼ), df = n-k-1
Excel 방법:
① 데이터 분석 → 회귀 분석
② X입력 범위: X1~X7 전체 열 선택
③ ⚠️ 주의: X₂(순자산), X₆(거래대금)은 LN() 변환 후 사용
④ 출력에서 R² · 수정R² · F통계량 · 각 계수 p값 확인
📏 독립표본 t-검정 — 공식 & 엑셀
H₀: μ₁ = μ₂ (두 집단 평균 같다)
H₁: μ₁ ≠ μ₂ (양측검정)

[등분산 가정]
합동분산: Sp² = [(n₁-1)s₁² + (n₂-1)s₂²] / (n₁+n₂-2)
t = (x̄₁-x̄₂) / (Sp·√(1/n₁+1/n₂))
df = n₁+n₂-2

[이분산, Welch t]
t = (x̄₁-x̄₂) / √(s₁²/n₁ + s₂²/n₂)
df ≈ (s₁²/n₁+s₂²/n₂)² / [(s₁²/n₁)²/(n₁-1)+(s₂²/n₂)²/(n₂-1)]
Excel 방법:
① 데이터 분석 → t검정: 등분산/이분산 가정 두 집단
② 먼저 F-검정(이분산검정)으로 등분산 여부 확인
③ p(F<=f) 양측 검정값이 0.05 이상이면 등분산 → 등분산 t검정
④ t값과 p값 확인: P(T<=t) 양측 < 0.05면 유의한 차이
📐 일원분산분석 (ANOVA) — 공식 & 엑셀
k개 집단, 각 nⱼ개 관찰값 (전체 N개)

SS분해:
SS합계 = ΣΣ(yᵢⱼ-ȳ)² = Σ(yᵢⱼ²) - N·ȳ²
SS처리(집단간) = Σnⱼ(ȳⱼ-ȳ)²
SS오차(집단내) = SS합계 - SS처리

F비 = MS처리/MS오차
MS처리 = SS처리/(k-1)
MS오차 = SS오차/(N-k)

효과크기: η² = SS처리/SS합계
해석: 0.01=작음, 0.06=중간, 0.14=큼
Excel 방법:
① 데이터 분석 → 분산 분석: 일원 배치법
② 입력 범위: 각 집단 열(그룹 형태) 또는 행 선택
③ 유의 수준: 0.05
④ 출력에서 요약(집단 통계)과 분산분석 표 확인
⑤ F 임계값보다 F가 크거나 P값 < 0.05면 유의

🔍 로그 변환 (Log Transformation)

X₂(순자산총액)과 X₆(평균거래대금)은 1억원~수십조원 범위로 극단적으로 오른쪽 편포되어 있습니다. 자연로그(ln)를 취하면 분포가 정규분포에 가까워지고 회귀계수 해석도 자연스러워집니다.
원래값: 100억 = 10,000,000,000원
ln변환: ln(10,000,000,000) ≈ 23.03

ln변환 계수 해석: X₂가 1단위 증가(e배 증가) → Y가 β₂만큼 변화
또는: X₂가 1% 증가 → Y가 β₂/100만큼 변화 (탄성치 해석)

💡 p값과 유의수준 해석 가이드

p값 범위표기해석
p < 0.001***매우 강한 증거 (0.1% 미만 확률로 우연)
0.001 ≤ p < 0.01**강한 증거 (1% 미만 확률로 우연)
0.01 ≤ p < 0.05*유의한 증거 (5% 미만 확률로 우연)
p ≥ 0.05ns유의하지 않음 (우연일 가능성 5% 이상)
